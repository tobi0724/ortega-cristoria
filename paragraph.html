Paragraph of HTML



In HTML, the <p> tag is used to define a paragraph of text. Paragraphs help organize content into readable chunks, making it easier for users to understand and interact with the page. The text inside a <p> tag is typically displayed with a margin before and after it, creating space between the paragraphs.


Basic Syntax of Paragraph in HTML:

  1. Text Content

    The <p> tag is designed to hold text content. It's used to break up the text into logical paragraphs.

  2. Line Breaks

    The <p> tag automatically adds space above and below the paragraph. You do not need to add line breaks (<br>) unless you want to control the flow of text within the paragraph itself.

  3. Nesting

    Paragraphs cannot be nested within each other. If you try to place a <p> tag inside another <p> tag, it will be automatically closed, and the browser will correct the markup by ending the first <p> and starting a new one.

  4. Inline Elements Inside Paragraphs

    You can include inline elements like links (<a>), images (<img>), bold text (<strong>), and other text formatting elements inside a <p> tag.

Key Points to remember

  • The <p> tag is used to define paragraphs, and it is a block-level element.
  • Paragraphs are typically separated by a margin and will automatically add space before and after the text.
  • Inline elements can be used within a paragraph for additional formatting (e.g., links, images, text formatting).
  • Paragraphs cannot be nested inside other paragraphs.
