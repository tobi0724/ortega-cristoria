CSS Margins

CSS margins create space around elements, outside of their borders.

Margin Properties:

  • margin-top – space above an element
  • margin-right – space to the right
  • margin-bottom – space below
  • margin-left – space to the left
  • margin – shorthand for setting all margins in one line

Example:


          div {

            margin: 20px;

          }

You can also use different values like auto, percentages, or specific units like em, rem, or px.
