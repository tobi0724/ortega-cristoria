What is a Favicon?



Favicon is a small image or icon that represents a website in the browser's tab or bookmark bar. It helps users identify a website easily among many open tabs. Favicons are set using the <link> tag within the <head> section of the HTML document.


How to Set a Favicon:

  • link rel="icon" - The link tag is used to define a favicon in HTML.
  • href - Specifies the path to the favicon file.
  • type - Defines the type of the image (optional but recommended, e.g., image/png).

Example:


    <link rel="icon" href="favicon.ico" type="image/x-icon">
