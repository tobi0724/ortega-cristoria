Elements of HTML


What is an element?

An HTML element is a fundamental building block of HTML(HyperText Markup Language) that is used to structure content on a webpage.


An element typically consists of:

  1. Opening Tag: The tag that marks the start of an element.
  2. Content: The actual content or text enclosed within the element.
  3. Closing Tag: The tag that marks the end of the element.

Example:

  • Opening Tag: <p>
  • Content: This is a paragraph
  • Closing Tag: </p>

  • <img> Embeds an image.

  • <br> Inserts a line break in text.

  • <hr> Creates a horizontal rule (a line).

  • <input> Creates an input field for forms.

Basic examples of HTML elements
Here are some of the most common HTML elements, along with examples:

  1. <h1> to<h6> - Headings

    Headings are used to define titles and subtitles on the page. There are six levels of headings, <h1> being the most important and largest, while <h6> is the least important and smallest.

  2. <p> - Paragraph

    The <p> tag is used to define a paragraph of text.

  3. <a> - Anchor link

    The <a> tag is used to define hyperlinks, which allow users to navigate to other pages or websites.

  4. <img> - Image

    The <img> element in HTML is used to embed images into a webpage.

  5. <ul> and <ol> - Unordered and Ordered lists Lists are used to group multiple items together.

    There are two types of list:

    • Unordered list <ul>: Items are displayed with bullet points.
    • Ordered list <ol>: Items are displayed with numbers or letters.

  6. <div> - Divisions

    The <div> tag is a block-level element used to group and structure content. It is often used with CSS for layout purposes.

  7. <span> - Inline Container
  8. <form> - Form
  9. <table>,<tr>,<td>,<th> - Tables
